# State-management-
# JavaScript Logic & State Management

## Project Description

This project is an Advanced To-Do List Application developed using HTML, CSS, and JavaScript.

The application demonstrates:

- DOM Manipulation
- Event Handling
- Event Delegation
- CRUD Operations
- Local Storage
- State Management
- Dynamic Rendering
- Task Filtering

## Features

### Create Task
Users can add new tasks.

### Read Tasks
Tasks are displayed dynamically.

### Update Task
Existing tasks can be edited.

### Delete Task
Tasks can be removed.

### Complete Task
Tasks can be marked completed.

### Local Storage
Tasks remain available after page refresh.

### Filtering
- All Tasks
- Active Tasks
- Completed Tasks

## Technologies Used

- HTML5
- CSS3
- JavaScript ES6

## Learning Outcomes

- DOM Manipulation
- Event Delegation
- State Management
- Local Storage API
- CRUD Operations
- Dynamic UI Rendering

## Expected Outcome

A fully functional task management application that retains user data across browser reloads.
